🚗 Real-Time Vehicle Number Plate Detection System

An end-to-end AI-powered Full Stack Application that detects vehicle number plates in real-time, extracts text using OCR, and stores structured data with timestamps.

🔥 Project Overview

This project is a real-world implementation of Automatic Number Plate Recognition (ANPR) — a system widely used in traffic monitoring, toll collection, and smart surveillance .

Unlike basic tutorial projects, this system is built as a complete pipeline:

🎥 Live video stream input (Camera / RTSP)
🧠 AI-based number plate detection
🔤 OCR-based text extraction
🗄️ Database storage with timestamps
📊 Frontend dashboard for visualization
🚀 Features

✔️ Real-time vehicle detection from live stream
✔️ Accurate number plate extraction using OCR
✔️ Stores every detection with timestamp (even repeated vehicles)
✔️ Smart duplicate handling logic
✔️ Live dashboard UI (React)
✔️ Backend API integration (Node.js / Express)
✔️ Database integration (SQL / SQLite)
✔️ Optimized for edge devices (Jetson Nano)
